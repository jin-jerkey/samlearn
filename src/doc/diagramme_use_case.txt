LÉGENDE:
--------
[ ] : Groupe de fonctionnalités
*   : Cas d'utilisation
|   : Relation/Association
+   : Acteur ou système

DESCRIPTION DES ACTEURS:
----------------------

1. ÉLÈVE:
   - Utilisateur principal qui suit les cours
   - Interagit avec le contenu pédagogique
   - Utilise l'assistant IA pour l'apprentissage
   - Suit sa progression et passe les examens

2. FORMATEUR:
   - Crée et gère le contenu pédagogique
   - Suit les progrès des élèves
   - Interagit avec les élèves via commentaires
   - Utilise l'IA pour optimiser ses cours

3. ADMINISTRATEUR:
   - Gère les utilisateurs et les droits
   - Supervise le système
   - Assure la maintenance
   - Contrôle les accès

RELATIONS PRINCIPALES:
-------------------

1. Élève → Cours:
   - Inscription
   - Consultation
   - Progression
   - Évaluation

2. Formateur → Cours:
   - Création
   - Modification
   - Gestion
   - Suivi

3. Administrateur → Système:
   - Administration
   - Surveillance
   - Maintenance
   - Sécurité

4. Tous → IA:
   - Assistance
   - Personnalisation
   - Analyse
   - Recommendations
